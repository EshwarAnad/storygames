    \n The forum software underwent an update that breaks the old link format. To get a new-style link from the old-style link, perform the following substitution:

    First, find the thread ID number from the old URL; it's carried in the "DiscussionID" variable:
    http://www.story-games.com/forums/comments.php?DiscussionID=5945&page=1#Item_15

    Then, plug that number into the following template:
    http://www.story-games.com/forums/discussion/[ID number]/

    You should end up with a valid thread address that way.
